# 공식 노트북 보관소

| 노트북 | 설명 |
|--------|------|
| `baseline.ipynb` | 공식 RandomForest baseline. LDAPS/GFS 격자 평균과 calendar feature를 사용해 그룹별 모델을 학습하고 제출 CSV를 생성합니다. |
| `evaluation_metric.ipynb` | 공식 평가 산식 예시 코드. actual capacity 10% 이상 구간에서 1-NMAE와 FICR을 계산합니다. |

이 폴더의 노트북은 참조 원본입니다. 실험·수정본은 `notebooks/` 또는 `src/baram/`에 별도 생성합니다.
