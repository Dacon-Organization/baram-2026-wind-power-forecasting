# 문제 정의

## 한 문장 정의

기상예보 데이터와 학습 기간의 발전량·SCADA 실측을 사용해 2025년 8,760시간의 KPX 3개 그룹별 풍력 발전량을 예측하고, 평균 오차와 정산금 획득률을 동시에 높이는 문제입니다.

## 예측 대상

| 대상 | 설비용량 | 평가 capacity |
|------|---------:|--------------:|
| `kpx_group_1` | 21.6 MW | 21,600 kWh |
| `kpx_group_2` | 21.6 MW | 21,600 kWh |
| `kpx_group_3` | 21.0 MW | 21,000 kWh |

## 입력 데이터의 본질

- LDAPS: 단지 주변 고해상도 예보 격자 16개.
- GFS: 광역 예보 격자 9개.
- SCADA: 학습 기간 터빈별 10분 단위 power, wind speed, wind direction.
- Label: 학습 기간 KPX 그룹별 시간 단위 발전량.
- Test: 2025년 LDAPS/GFS만 제공. 평가 기간 SCADA와 실제 발전량은 제공되지 않습니다.

## 평가 문제의 본질

1. 기상예보 격자를 발전단지의 풍력 조건으로 후처리합니다.
2. 풍속·풍향·기압·온도·공간 차이를 그룹별 발전량으로 변환합니다.
3. 오차율 6%/8% 구간을 의식해 FICR을 개선합니다.
4. 예측 기준시점 이후 정보를 쓰지 않아야 합니다.

## 성공 기준

| 단계 | 성공 기준 |
|------|-----------|
| 1차 리더보드 | Private score 상위권, 제출 CSV 형식 오류 0건 |
| 코드 검증 | Private score 오차 범위 내 재현, train/inference 분리 |
| 발표 평가 | 과제 이해도, 기술 우수성, 문제 해결력, 적용 가능성 설명 |
| 운영 | 제출 파일 버전, hash, commit, seed, config 복구 가능 |
