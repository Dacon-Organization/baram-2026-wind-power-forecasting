# 2026 BARAM 풍력발전량 예측 AI 경진대회

> 한국동서발전 · GS E&R · 태백가덕산풍력발전 주최/주관, 데이콘 운영
> **핵심 과제: 기상예보 데이터 기반 KPX 그룹별 시간 단위 풍력발전량 예측**

이 프로젝트는 `제3회 풍력발전량 예측 AI 경진대회 - BARAM 2026` 참가를 위한
문제 정의, 데이터 설계, 모델링, 제출 운영, 산출물 검증 자료를 관리합니다.

2026-07-06 공식 데이터 공개 전까지는 사전 워크숍 자료와 공식 안내를 기반으로
가설을 세우되, 모든 데이터 관련 항목은 `확정 / 판독 불확실 / 공식 공개 후 재검증`
상태를 분리해 관리합니다.

---

## 1. 대회 개요

| 항목 | 내용 |
|------|------|
| 대회명 | 제3회 풍력발전량 예측 AI 경진대회 - BARAM 2026 |
| 주제 | 기상예보 데이터 기반 풍력발전량 예측 AI 모델 개발 |
| 주최/주관 | 한국동서발전, GS E&R, 태백가덕산풍력발전 |
| 운영 | 데이콘 |
| 참가자격 | 국내·외 대학생 및 대학원생, 졸업예정자 및 취업준비생 |
| 사용 가능 언어 | Python |
| 대회 기간 | 2026-07-06 10:00 ~ 2026-08-14 10:00 |
| 1차 제출물 | 모델 예측 결과 CSV |
| 2차 제출물 | train/inference 분리 코드, 모델, 외부데이터, 발표 PDF, 자격 증빙 |

공식 안내: [Dacon 대회 페이지](https://dacon.io/competitions/official/236727/overview/description)

## 2. 현재 확정한 작업 순서

1. **전략 PRD**
   문제 정의, 시장/제도 맥락, 평가식, 데이터 팩트 레저, 실격 방지 원칙을 먼저 고정합니다.

2. **데이터·모델링 설계서**
   공식 데이터 공개 직후 스키마 검증, 피처 설계, 검증 분할, 모델 후보, 실험 추적 체계를 확정합니다.

3. **운영 마스터플랜**
   제출 일정, 리더보드 운영, 산출물 재현성, 2차 발표 준비, 역할 분담을 관리합니다.

## 3. 핵심 데이터 팩트

| 항목 | 현재 상태 | 내용 |
|------|-----------|------|
| 예측 대상 | 확정 | `KPX Group 1`, `KPX Group 2`, `KPX Group 3` |
| 그룹 용량 | 확정 | 21.6MW / 21.6MW / 21.0MW |
| 터빈 수 | 확정 | Group 1: 6기, Group 2: 6기, Group 3: 5기 |
| 터빈 모델 | 확정 | Group 1/2: VESTAS V126, Group 3: UNISON U136 |
| 총 설비용량 | 확정 | 64.2MW |
| 총 터빈 수 | 확정 | 17기 |
| 그룹 위치/지명 | 판독 불확실 | 지명은 단정하지 않고 색상 기반 그룹-터빈 매핑만 사용 |
| 학습 기상 데이터 | 확정 | 2022-2024 LDAPS, GFS |
| 학습 label | 확정 | Group 1/2: 2022-2024, Group 3: 2023-2024 |
| 평가 기상 데이터 | 확정 | 2025 LDAPS, GFS |
| 평가 SCADA/KPX label | 확정 | 제공되지 않음 |
| 예측 기준시점 | 확정 | D-1 14:00 KST |
| 예보 리드타임 | 워크숍 근거 | 00UTC 기준 `h016~h039` 제공으로 판독 |

상세 근거표는 [전략 PRD](docs/groundwork/01-strategy-prd.md)의 `데이터 팩트 레저` 섹션을 기준으로 합니다.
